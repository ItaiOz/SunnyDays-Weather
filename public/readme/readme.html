Execution

The submission is a WebStorm project that can be run directly from the IDE.

Assumptions

The site use bootstrap CDN therefore assumes an internet connection is available.

The program is using nre methods that we have learned in last classes.

We are communicating with server which is our local host, routing between pages and practicing methods.

I created a register, login and password pages which we navigate between them until we are inside the weather website

When we are in the website after we logged in we are using get, post and delete method to update the list

Most of the validation happens on server side. When I am modifying the table it is done by the server

and the client is getting back a response. Therefore, in every request that is sent I am resetting the data

on client side abd getting response from server and editing the locations list.

The navbar on the top of the home page allows us to navigate to different pages:

Clicking on the big O-W button will just show us the logo of the website

Clicking on Home button will actually guide us back the weather page since we want to avoid collisions

Sign out will leave to login page, and while we are there we will turn the login flag off so we will have

to login again. The readme file leads here :)
